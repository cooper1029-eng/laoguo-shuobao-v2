# 🏺 老郭说宝 — 创作工作台 V2

纯前端网页，双击即用。专注建水紫陶，7步完成一篇文章。

## 快速开始

1. **打开页面**：双击 `index.html`
2. **配 API Key**：点右上角 ⚙️，填入你的 DeepSeek API Key
3. **推荐选题**：点「AI 推荐选题」→ AI 会基于你的知识库策划选题
4. **跟着流程走**：选题 → 识图 → 正文 → 修改 → 标题 → 配图 → 导出

## 识图功能

上传器物图片后，点击「开始识图」即可。

### 方式一：通过 Python 代理（推荐，识别更准）

```bash
cd 老郭网页版V2
python3 proxy/vision_proxy.py
```

代理启动后，网页端的识图请求会自动通过它转发到 oMLX。保持终端开着即可。

> 需要安装 `requests` 库：`pip3 install requests`（通常已自带）

### 方式二：手动输入

如果不想启动代理，识图失败后会自动弹出输入框，你自己写下器物描述就行，不影响后续流程。
